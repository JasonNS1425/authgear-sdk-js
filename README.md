## Running the example

```sh
$ git clone --branch next https://github.com/SkygearIO/skygear-SDK-JS.git
$ cd skygear-SDK-JS
$ npm install
$ npm run lerna bootstrap
$ npm run build
$ SKYGEAR_ENDPOINT=<endpoint> SKYGEAR_API_KEY=<apikey> npm run example
```

## Contributing
